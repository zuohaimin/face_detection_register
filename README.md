##基于人脸识别算法的登陆验证系统  
*** 
这个项目是来自本科大学毕业设计项目，旨在将人脸识别算法结合到登陆验证系统中。

项目的主要功能：
1. 基于人脸识别的登陆与注册
2. 管理员可以自由配置基于 用户-角色-资源 的权限关系
3. 人脸采集信息的汇总展示（主要是包括年龄分布图，性别比例图，脸型比例图，颜值曲线图）
